# Improv 
Improv is an app that helps people to organize things they have to say during a presentation and helps people to rehearse before the presentation

**Improv's features:**
* CRUD Notes
* Record audio
* Play recorded audio

**Frameworks/Tech that I use:**
* UIKit
* CoreData
* FileManager
* AVFoundation
